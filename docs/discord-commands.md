# Discord Commands

## Ticket setup and lifecycle

- `/setup_tickets`
- `/ticket_panel`
- `/close_ticket`
- `/reopen_ticket`
- `/ticket_info`

## Assignment

- `/assign_ticket`

Inside a ticket thread:

- no user passed assigns the ticket to yourself
- providing a user assigns it to that staff member

## Tags

- `/list_tags`
- `/create_tag`
- `/edit_tag`
- `/delete_tag`
- `/add_ticket_tag`
- `/remove_ticket_tag`

Managed tags are admin-defined and can also be maintained from the dashboard Admin page.

## User/thread management

- `/add_ticket_user`

## Permissions

Discord bot permissions:

- View Channel
- Send Messages
- Read Message History
- Create Public Threads
- Send Messages in Threads
- Manage Threads
- Embed Links
- Attach Files
- Manage Messages

Discord Developer Portal privileged intents:

- Presence Intent
- Server Members Intent
- Message Content Intent
